CliTrackCV

Chats autónomos para almacenar fotos de documentos por cliente

Información de Desarrollo

Esta aplicación fue desarrollada con Android Studio como entorno de desarrollo, utilizando Java como lenguaje de programación principal. Diseñada para crear chats individuales donde almacenar fotos de documentos organizados por cliente.

Android Studio
Java
Android SDK

Desarrollado por Carlos Vega

Interfaces de la Aplicación

1

Lista de Clientes

La interfaz principal presenta una lista completa y organizada de todos los clientes registrados en el sistema. Cada cliente se muestra con su información básica y estado de comunicación.

2

Registro de Nuevos Clientes

Interfaz simple y eficiente para el registro de nuevos clientes. Permite capturar la información esencial de manera rápida y organizada para mantener la base de datos actualizada.

  • Integración directa con la lista principal
3
